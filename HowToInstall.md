## STEP 1 - Dependencies
> You need to have this funny tool on your pc.
- [NodeJS +22.0](https://nodejs.org/en/download)

## STEP 2 - Verify Installation
> Verify if the node is installed using this command
```bash
node -v
```
> Verify if npm (package manager) is installed using this command
```bash
npm -v
```

## STEP 3 - Install and execute!
> Access a terminal/command prompt in the same directory that contains `package.json`. Run the command:
```bash
npm install
```
> To install dependencies, to run execute this command:
```bash
npm run start
```
> To build an executable, run a command containg your OS:
```bash
npm run build:win
npm run build:mac
npm run build:linux
```
> Extra: if you want to build a portable app
```bash
npm run build:portable
```
